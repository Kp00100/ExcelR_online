SELECTOR

Universal Selector

it indicates with *

it select all elements

Grouping selector

It is used to select group of elements
It is indicated with commaa(,)

Combined Selector

  1. descendent selector

    it selects all child elements(direct/indirect)

                        parent child{
                        // both direct and indirect childs will select
                    }

  2. child selector

    it selects direct child selector

                        parent > child{
                        //it selects direct child
                    }

  3. adjacent sibling selector

    it select immediate adjacent element

                        main +adjacentelementd{
                        //it select immediate adjacent element
                    }

  4. general sibling selector

    it selects all adjacent elements

                        main ~adjacentelementd{
                        //it selects all siblings to main elements
                    }
