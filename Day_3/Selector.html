Selector

Selector is used to select the elements in the html

Selector is used to apply css to the selected elements

Selector is used to apply js to the selected elements

Types of Selectors

  1. Element Selector
  2. Id Selector
  3. Class Selector
  4. Attribute Selector
  5. Universal Selector
  6. Group Selector
  7. Descendent Selector
  8. Child Selector
  9. Adjacent Selector
  10. General Selector

Element Selector

Element Seector is used to select the elements based on the tag name

Class Selector

  • Class Selector is used to select elements based on the class name
  • Class Selectore not to be unique
  • We can use class selector with .className for css style

Id Selector

  • ID Selector is used to select elements based on id name
  • id slector must be unique
  • we can use id selector with #idName for css style

Rules:

  1. class/id name should have alphabets and numbers
  2. class/ id name not to start with number
  3. className and idName can be same
